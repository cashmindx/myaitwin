🚀 Yoco Payment Integration Guide for My AI Twin

Step 1: Create Yoco Account

1. Visit yoco.com

2. Sign up for a business account

3. Complete KYC verification

4. Get your API keys from the dashboard

Step 2: Environment Variables

Add these to your .env file:

YOCO_SECRET_KEY=sk_test_your_secret_key_here YOCO_PUBLIC_KEY=pk_test_your_public_key_here YOCO_WEBHOOK_SECRET=whsec_your_webhook_secret_here

Step 3: Install Yoco SDK

npm install @yoco/yoco-sdk-web

Step 4: Frontend Integration

Add this to your payment component:

import { YocoSDK } from '@yoco/yoco-sdk-web'; const yoco = new YocoSDK({ publicKey: process.env.REACT_APP_YOCO_PUBLIC_KEY }); const handlePayment = async (amount, currency = 'USD') => { try { const result = await yoco.showPopup({ amountInCents: amount * 100, currency: currency, name: 'My AI Twin', description: `${planName} Plan Subscription`, image: 'https://your-domain.com/logo.png', locale: 'en', allowRememberMe: false }); if (result.error) { console.error('Payment failed:', result.error); } else { // Send token to your backend await processPayment(result.id); } } catch (error) { console.error('Payment error:', error); } };

Step 5: Backend Processing

Create an API endpoint to process payments:

// /api/process-payment const yoco = require('yoco'); app.post('/api/process-payment', async (req, res) => { try { const { token, amount, plan } = req.body; const charge = await yoco.charges.create({ token: token, amountInCents: amount * 100, currency: 'USD', metadata: { plan: plan, userId: req.user.id } }); if (charge.status === 'successful') { // Update user subscription await updateUserSubscription(req.user.id, plan); res.json({ success: true, chargeId: charge.id }); } else { res.status(400).json({ error: 'Payment failed' }); } } catch (error) { res.status(500).json({ error: error.message }); } });

Step 6: Webhook Setup

Set up webhooks for payment confirmations:

// /api/yoco-webhook app.post('/api/yoco-webhook', (req, res) => { const signature = req.headers['x-yoco-signature']; const payload = JSON.stringify(req.body); // Verify webhook signature const expectedSignature = crypto .createHmac('sha256', process.env.YOCO_WEBHOOK_SECRET) .update(payload) .digest('hex'); if (signature === expectedSignature) { const event = req.body; switch (event.type) { case 'payment.succeeded': // Handle successful payment handleSuccessfulPayment(event.data); break; case 'payment.failed': // Handle failed payment handleFailedPayment(event.data); break; } res.status(200).send('OK'); } else { res.status(400).send('Invalid signature'); } });
